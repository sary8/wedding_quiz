## 対象: FinalPage（最終結果発表）のみ
## スタイル: オールスター感謝祭風バッチ発表

カウントダウン・各問題の結果発表・中間ランキングは現状維持。

---

## バッチ発表の流れ

全参加者を順位帯ごとのバッチに分け、下位から順に発表。
各バッチ内では名前が高速スクロールで現れ、全員出揃ったら一時停止。
**Top5はスポットライト演出（ホスト手動）。**

### バッチ構成（例: 50人参加の場合）

| バッチ | 順位帯 | 人数 | スクロール速度 |
|--------|--------|------|---------------|
| 1 | 50位〜31位 | 20人 | 高速（1行 150ms） |
| 2 | 30位〜11位 | 20人 | 高速（1行 150ms） |
| 3 | 10位〜6位 | 5人 | 低速（1行 800ms） |
| — | 5位〜1位 | 5人 | ホスト手動スポットライト |

**バッチ分割ルール:**
- 6位以下: 20人ずつのバッチ（端数は最初のバッチ）
- 10位〜6位: 最後のバッチ
- 5位〜1位: バッチではなくスポットライト個別演出

※参加者が20人以下の場合はバッチを減らす（6位以下が1バッチのみ等）

## 画面レイアウト

### 表示項目（全画面共通）
各行に以下4項目を必ず表示:
- アイコン（アバター画像 or イニシャル）
- ニックネーム
- 合計スコア
- 平均回答時間

## バッチの挙動詳細

### 各バッチの流れ
1. 画面が空の状態から開始
2. 行が**下から1行ずつ出現**し、上に積み上がっていく（最下位から）
3. 全行が表示されたら**2秒停止**
4. フェードアウト → 次のバッチへ

### スクロール速度
| 順位帯 | 1行あたり | 備考 |
|--------|-----------|------|
| 21位以下 | 150ms | 高速。名前がさっと流れる |
| 10位〜6位 | 800ms | 低速。1人ずつ注目される |

## Top5 スポットライト（ホスト手動）

バッチ発表で6位まで表示した後、Top5は**ホストがボタンを押すたびに1人ずつ発表**。

### ホスト画面
```
┌──────────────────────────────────────┐
│ [次の順位を発表] ボタン              │  ← ホストが手動で押す
│                                      │
│  （スポットライト演出表示中）         │
└──────────────────────────────────────┘
```

### 各順位の演出（2段階）

**ステージ1: じらし**
- 画面中央に「第○位は…」テキスト（text-7xl）
- テキスト微振動（x: [-2, 2], 150ms周期）
- ドラムロール効果音（既存 playDrumRoll）
- 背景がメダル色/アクセント色へ徐々に変化

| 順位 | じらし時間 |
|------|-----------|
| 5位 | 1.5秒 |
| 4位 | 2秒 |
| 3位 | 2秒 |
| 2位 | 2.5秒 |
| 1位 | 3.5秒 |

**ステージ2: 発表**
- アバターが scale 0→1 でズームイン（spring, stiffness:120）
- 名前が y:30→0 でスライドアップ（delay:0.3s）
- スコア + 平均回答時間がフェードイン（delay:0.6s）
- ファンファーレ + 紙吹雪

| 順位 | 紙吹雪 |
|------|--------|
| 5位 | なし |
| 4位 | なし |
| 3位 | ブロンズ色 50個 |
| 2位 | シルバー色 100個 |
| 1位 | ゴールド色 200個 × 3回 + 追加2回（計5回） |

**1位追加演出:**
- 画面フラッシュ（白→透明, 200ms）
- 背景全体が金色グラデーションにアニメーション変化

### スポットライト画面の表示項目
```
┌──────────────────────────────────────┐
│                                      │
│            第3位                      │  ← 大きく順位
│                                      │
│         [アバター 192px]             │  ← 大きなアバター
│                                      │
│          けんじ                       │  ← 名前
│        1,250点                       │  ← スコア
│      平均回答: 1.85秒                │  ← 平均回答時間
│                                      │
└──────────────────────────────────────┘
```

### ホスト → Display 連動
- ホストが「次の順位を発表」ボタンを押す
- Socket.io で `revealNextRank` イベントを送信
- Display画面が自動で次の順位を表示

## 技術的メモ

- FinalPage.tsx の scroll/top3 フェーズを全面刷新
- teamReveal, group フェーズは現状維持
- Top5手動発表: HostPage から Socket.io イベント or コールバックで制御
- DisplayPage も同じ演出を isDisplay=true で表示（ボタン非表示）
- スクロールは行の出現で実装（translateY ではなく1行ずつ追加）
- `prefers-reduced-motion` 時はバッチごとに即時全表示
- 効果音は既存 Web Audio API 範囲内（playDrumRoll, playFanfare）
